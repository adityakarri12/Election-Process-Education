## 🚀 Key Features

*   **Electoral Intelligence Hub**: Real-time tracking of upcoming elections, live results, and past outcomes across India.
*   **Constituency Pulse**: Instant mapping of any 6-digit Indian Pincode to its representatives with **Google Maps Booth Finder**.
*   **Electra AI Chatbot**: A professional electoral analyst powered by Gemini for answering complex democratic questions.
*   **ID Verification Simulator**: AI-powered document analysis using **Google Cloud Vision AI** for educational verification.
*   **Live Expert Analytics**: Firestore-backed global leaderboard of top democratic participants.
